# CSE111 Lab Exercises

This repository contains all lab tasks from the CSE111 course, originally in a Jupyter notebook, now split into separate Python files.

Each file corresponds to a lab session (LAB-1 to LAB-8) and includes the problem statements as comments before each solution.

The `problems.py` file contains additional practice problems from the notebook, including "PROBLEM", "LAB mid", and "Final" exam questions.

## How to run
```bash
python lab1.py
